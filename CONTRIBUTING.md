This project is a modified version of GeyserMC. All contributions made to GeyserMC software should be submitted on the official build at this time since this project is in it's Alpha stage. Once this project is released contributions may be made to our version of GeyserMC. 

__Future Contribution Guidelines__

Contributions through the form of pull requests are welcome on this project. Pull requets should be submitted to fix issues with this build of GeyserMC. Issues with the main build of GeyserMC that do not affect our build should not have pull requests submitted here to fix them. Instead submit the pull request under GeyserMC official build. New features submitted through pull requests are always welcome. More information on the Contributiuon Guidelines will be added as we move closer to our official release.
